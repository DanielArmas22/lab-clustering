# Generación de Gráficos de Distribución

Este directorio contiene el código necesario para generar visualizaciones de distribución sobre el dataset de estudiantes.

## Archivos
- `generate_distributions.py`: Script principal en Python.

## Ejecución
Para generar los gráficos, ejecuta desde la terminal:
```bash
python generate_distributions.py
```

Los gráficos se guardarán automáticamente en una subcarpeta llamada `img/`.

## Requisitos
- pandas
- matplotlib
- seaborn
