# Visualizaciones Dataset 02

Este directorio contiene herramientas para el análisis exploratorio visual del dataset `Estudiantes.csv`.

## Contenido

- `generate_distributions.py`: Genera análisis de las notas (Matemáticas, Lectura, Escritura) y perfiles demográficos.

## Cómo usar

Ejecuta el script desde la carpeta `charts`:

```bash
python generate_distributions.py
```

Las imágenes se guardarán en:
`charts/img/`

## Variables Analizadas

- **Numéricas:** Matematicas, Lectura, Escritura.
- **Categóricas:** Genero, Etnia, Educación de los padres, Preparación del examen.
